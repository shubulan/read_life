# 检视阅读

## 分类
实用型书籍

## 一句话总结
作者认为通过提问的方式可以帮助我们解决问题。并给出了我们在生活正四个典型的场景下能够提出的几百个好问题。这四个场景分别式：决策、创新、沟通、管理。并且解释了为什么这些问题为什么是好问题。

## 作者想要解决的主要问题
1. 为什么我们能通过提问来解决问题？
2. 在决策、创新、沟通、管理的场景下，我们能通过提出哪些好问题？以及为什么这些是好问题？
